在主机控制台绑定域名
将 LIKLUKTECH.COM 绑定到您购买的主机空间
  1. 登录您购买的主机控制台（cPanel / 腾讯云控制台 / 阿里云控制台）
  2. 找到 域名绑定 或 添加域名 选项
  3. 填入域名：likluktech.com（和 www.likluktech.com）
  4. 确认绑定，系统会给您一个 主机 IP 地址（形如 123.45.67.89），记下来备用
⚠️ 注意 绑定域名后，主机会给您分配一个 IP 地址（或 CNAME 地址），这是下一步 DNS 解析需要用到的关键信息。
3
设置 DNS 域名解析
在域名注册商处将域名指向主机 IP

登录您购买域名的平台（GoDaddy / 阿里云 / 腾讯云 / Namecheap 等），进入 DNS 管理，添加以下解析记录：

# A 记录 — 主域名解析（必须添加）
类型（Type）： A
主机名（Name）： @ （代表根域名 likluktech.com）
值（Value）： 填写主机给您的 IP 地址
TTL： 600（或默认值）

# A 记录 — www 子域名解析（建议添加）
类型（Type）： A
主机名（Name）： www
值（Value）： 填写主机给您的 IP 地址（同上）
TTL： 600
💡 DNS 生效时间 DNS 解析一般需要 1～24 小时生效，全球扩散最长 48 小时。国内通常 10 分钟内即可生效。
💡 如果主机给的是 CNAME 地址而非 IP 则添加 CNAME 记录：
类型：CNAME · 主机名：@ 或 www · 值：主机提供的 CNAME 地址

上传网站文件
通过 FTP 或文件管理器将文件上传到主机

有两种方式可以上传网站文件，选择您习惯的方式：

方式一：使用文件管理器（最简单）

  1. 登录主机控制台，找到 文件管理器（File Manager）
  2. 进入网站根目录（通常是 public_html 或 www 或 htdocs 文件夹）
  3. 点击 上传，上传整个 likluktech-website.zip 文件
  4. 上传完成后，在文件管理器中 解压 这个 ZIP 文件
  5. 确保 index.html 文件直接在根目录下（不要多一层文件夹）
⚠️ 重要：检查目录结构 解压后正确的结构应该是：
public_html/（主机根目录）
  ├── index.html（首页，必须在这里）
  ├── products.html
  ├── about.html
  ├── contact.html
  ├── style.css
  ├── main.js
  └── images/
     ├── transmission/（产品图片）
     ├── gearbox/（变速箱图片）
     └── whatsapp-qr.jpg

方式二：使用 FTP 软件（推荐用 FileZilla）

  1. 下载安装 FileZilla（免费，官网 filezilla-project.org）
  2. 从主机控制台获取 FTP 账号信息（主机、端口、用户名、密码）
  3. 打开 FileZilla，在顶部输入 FTP 信息，点击 快速连接
  4. 连接成功后，左侧是您的电脑，右侧是主机服务器
  5. 右侧进入 public_html（或 www）目录
  6. 左侧找到 C:\Users\THINK09\WorkBuddy\20260329093252\likluktech\ 文件夹
  7. 选中 likluktech 文件夹里的所有文件，拖到右侧上传
💡 FileZilla FTP 填写示例
主机（Host）：ftp.likluktech.com 或主机 IP
用户名：主机提供的 FTP 用户名
密码：主机提供的 FTP 密码
端口：21（默认）

上线后建议做的事 可选但重要
帮助网站被更多海外客户找到

    大多数主机提供免费的 Let's Encrypt 证书，在控制台一键安装。HTTPS 对 Google 排名很重要，外贸客户也更信任 HTTPS 网站。

    访问 search.google.com/search-console，添加您的网站，提交 sitemap，让 Google 更快收录。
  3. 注册 Google Business
    在 Google 商家档案上建立页面，有助于买家搜索时找到您。
  4. 配置 sales@likluktech.com 邮箱
    在主机控制台创建企业邮箱账号，用于接收客户询盘更专业。
  5. 定期更新产品内容
    网站内容越新鲜，搜索引擎排名越好。建议每月添加新产品或更新报价信息。

遇到问题时参考

Q：域名解析了但网站打不开？

A：DNS 需要等待 1-24 小时生效。可以用 dnschecker.org 查询解析是否已扩散。

Q：上传后打开首页是空白或 403 错误？

A：检查 index.html 是否在 public_html 根目录，而不是子文件夹里。

Q：产品图片不显示？

A：检查 images 文件夹是否完整上传，路径是否正确（images/transmission/xxx.png）。

Q：网站在手机上显示有问题？

A：网站已做响应式设计，如有问题请截图发给我，我来调整代码。

Q：想修改网站内容怎么办？

A：告诉我想改什么，我直接帮您修改 HTML 文件，然后只需重新上传修改过的文件即可。
